# Ruboty::GoogleImage
An ruboty handler to search images from Google.

## Install

This gem needs two enviroment variables.

- GOOGLE_CSE_ID
- GOOGLE_CSE_KEY

## Usage
```
@ruboty image <keyword> - Search image from Google
```
